Voting Mechanics

Plurality: Enabled - The player with the most votes after 48 hours will die. Majority: Enabled day 2+ - If a player is voted by more than half of the currently living players then the day immediately ends with that player’s death. NoLynch: Enabled day 2+ - If this option has the most votes at the end of the day then nobody dies. If this option is voted by over half the players then the day immedaitely ends with nobody dying. Ties: Resolved by RNG (except if No Lynch) - If two or more players are tied in most votes when the day ends, a random one is chosen to be eliminated. If one of the tied votes is in the No Lynch option with more than 3 players alive, however, it will take priority and no one will get lynched. No Votes: Resolved by RNG - If nobody is voted at the end of the day, a random player is eliminated.

Private Chat and Mafia Mechanics

The Mafia has a 24/7 private chat on their own private server. - The Mafia has access to multitasking. If the game includes Masons, they will also have a private chat on their own private server.

Activity Requirements

Minimum post count: - You are required to post at least 20 game related messages each 24 hours of a day phase or you will get prodded. - If you write 0 posts in a 24h period, you will receive 2 prods at once. - If you get 3 prods, you will be forced replaced. - Mod discretion may result in variation to these rules. Please only sign up for this game if you have time to play!

Rules

Please make sure to have read the Game Rules section of our #📜rules channel, so that you do not accidentally break one of them. Breaking our rules can lead to punishments, such as game bans, or forceful replacement by a different player, which I would like to avoid if possible.
