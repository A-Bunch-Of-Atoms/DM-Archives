Welcome to NE42, NewD3!

The following will be the rules, mechanics as well as the setup.

Phases

Starting Phase: Daystart Day: 48 Hours Night: 24 Hours Day Starting Time: At 20 UTC0 -> 18 UTC 0 = 2:00 p.m. in your timezone

Voting Mechanics

Plurality: Enabled - The player with the most votes after 48 hours will die. Majority: Enabled starting day 2. - If a player is voted by half of the currently living players +1 then the day immediately ends with that players death. NoLynch: Enabled starting day 2 - If this option has the most votes at the end of the day then nobody dies. If this option is voted by half of the living players +1, then the day immedaitely ends with nobody dying. Ties: Resolved by RNG - If two or more players are tied in votes when the day ends, a random one is chosen to be eliminated. No Votes: Resolved by RNG - If nobody is voted at the end of the day, a random player is eliminated.

Private Chat and Mafia Mechanics

Mafia/Masons have a 24/7 private chat. The Mafia has access to multitasking. The Mafia has a mandatory nightly factional kill.

Activity Requirements

Minimum post count: You are required to post at least 20 times each 24 hours of a day phase. - Mod discretion may result in variation however. Minimum content posts: You are required to post these 20 messages in relation to the game. - Posts that are not related to the game would be off-topic posts, or posts that don't contribute to the game in a meaningful way, such as small talk for example. Warnings: Each time you do not pass an activity check you will receive a warning, receiving a third warning will result in being removed from the game. Replacements: Each slot may only see one replacement total, after which it will instead be modkilled if another replacement becomes necessary. Please only sign up for this game if you are able to commit an adequate amount of time to it (1 hour or more each day).

Questions

Since this game is meant to be a learning experience, and specifically catering towards new players, I will try to help you out whenever I can, just ask me anywhere where I can see it, mandatorily in private if the game is ongoing.

Rules

Please make sure to have read the Game Rules section of our #📜rules channel, so that you do not accidentally break one of them. Breaking our rules can lead to punishments, such as game bans, or forceful replacement by a different player, which I would like to avoid if possible.
